# Password-checker

A simple Python programme that ensures a password entered meets all of the necessary rules.
This system asks the user to enter the password until all the conditions are met.

✅ Be at least 8 characters long

✅ Contain at least one uppercase letter

✅ Contain at least one lowercase letter

✅ Contain at least one digit (0–9)

✅ Contain at least one special symbol from: $, #, @

❌ Not be a common password (e.g., password, 12345, etc.)
